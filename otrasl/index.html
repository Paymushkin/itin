Стандартная Проектная компания + Битрикс24

В Стандартную установку решения входит: Настройка базового функционала Битрикс24,настройка работы с лидами, создание направлений работы со сделками в воронке продаж, создание автоматизированного рабочего Проектного отдела для ведения работы по проектам компании, настройка роботов и бизнес-процессов для автоматизации работы CRM.

Отраслевое решение CRM для компаний, занимающихся проектной деятельностью. Помогает организовать работу по ведению и разработке проектов. Руководители компании всегда могут получить информацию о состоянии проектов и их финансовом срезе.
Данное готовое решение подойдет, как для компаний, разрабатывающих проекты, так и для компаний ведущих полных цикл работы над проектом от разработки до реализации и надзора.

Программа предусматривает возможность интеграции с 1С, платежными системами, IP-телефонией, почтой через готовые модули или API.Настройка CRM

Продажи проектов

Обработка поступившего запроса на разработку проекта, подготовка коммерческого предложения (КП) по проекту, заключение договора на разработку и реализацию проекта, получение полной или частичной оплаты выполненных работ. 
Перевод проекта в стадию разработки, передача его в проектный отдел компании, для выполнения работ по разработке проекта. 
Финальной стадией процесса является получение полной оплаты за разработку и работы по проекту и, при необходимости, получение обратной связи по результатам выполненных работ. 

Для работы с заказами компании разработана специализированная карточка сделки с необходимыми для обработки заказа полями. Предусмотрена обязательность заполнения ключевых полей сделки при переходе на стадии обработки.
Для работы с карточкой заказа, кроме базовый полей, созданы специальные пользовательский поля для заполнения данных по проекту и характеристикам заказа клиента.

В карточке сделки реализовано подключение дополнительного функционала расчета предварительной оценки работ по проекту и данных разработки проекта.

Оценка работ по проекту

Данный инструмент предназначен для составления предварительного списка этапов разработки, работ по проекту, сроков и стоимости работ при составлении коммерческого предложения (КП) для заказчика, в случае если эти виды работ и этапов не были занесены как услуги в товарном каталоге CRM.

План-график разработки проекта

Составление и ведение План-графика работ по проекту с автоматическим отслеживание сроков выполнения запланированных этапов работ.Возможность создания связанных задач по утвержденному план-графику работ.

Документы по проекту

Подключения в карточку разработки ведения реестра документов по проекту. Позволяет получить доступ из карточки проекта ко всем необходимым документам и договорам по проекту с описанием списка работ, подрядчиков по работам и т.д.

Расходы и платежи

Фиксация в карточке проекта расходов и платежей по проекту (финансы по проекту), как запланированных, так и уже произведенных с автоматическим расчетом в поле карточки суммы текущих расходов по проекту, запланированных расходов по проекту и текущей доходности проекта.

HR отдел (отдел кадров)

Мониторинг выполнения строительных работ на объекте заказчика. При переходе сделки в стадию «Передачи в отдел Реализации» в соответствующем направлении обработки создается карточка проекта с автоматическим получением необходимых данных из сделки.
